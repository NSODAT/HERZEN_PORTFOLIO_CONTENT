# HERZEN_PORTFOLIO_CONTENT

Хранилище файлов работ по курсам для портфолио.

Структура:

- `1 курс/` — материалы 1 курса
- `2 курс/` — материалы 2 курса
- `3 курс/` — материалы 3 курса
- `4 курс/` — материалы 4 курса

Сайт портфолио: [HERZEN_PORTFOLIO](https://github.com/NSODAT/HERZEN_PORTFOLIO)

## Ссылки на файлы из портфолио

В `data/portfolio.json` можно указывать ссылки на файлы в этом репозитории:

`https://github.com/NSODAT/HERZEN_PORTFOLIO_CONTENT/blob/main/1 курс/имя-файла.pdf`
